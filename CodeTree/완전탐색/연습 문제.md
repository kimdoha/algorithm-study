## 연습 문제

### 이상한 진수 2
``` markdown
0 이상의 정수 N을 2진법으로 나타낸 뒤,
그 숫자들 중 정확히 한 숫자만을 바꾼 숫자 a가 주어졌을 때,
가능한 숫자 N 중 최댓값을 찾는 프로그램을 작성해보세요.
```

``` java
    static String binaryNum, changeNum;
    static int maxValue = Integer.MIN_VALUE;

    static int calculateToDecimal() {
        int decimalSum = 0;
        int index = changeNum.length() - 1;
        
        for(int i = 0; i < changeNum.length(); i++) {
            decimalSum += Math.pow(2, i) * (changeNum.charAt(index - i) - '0');
        }

        return decimalSum;
    }

    static void solution() {
        if(binaryNum.contains("0")){
            int index = binaryNum.indexOf("0");
            changeNum = binaryNum.substring(0, index) + "1" + binaryNum.substring(index + 1, binaryNum.length());
        } else {
            changeNum = binaryNum.substring(0, binaryNum.length() - 1) + "0";
        }
    }

```
